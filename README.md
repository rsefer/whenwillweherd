## Prerequisites

Install [Node.js](http://nodejs.org/) on your local machine.

## Development

```
npm install
npm start
```

To use HTTPS locally, set the following environmental variables:
```
export SSL_KEY_PATH=/path/to/server.key
export SSL_CRT_PATH=/path/to/server.crt
```
